## A. 中国：宏观、政策与监管（官方一手）
1) **中国人民银行（PBoC）**
- 内容：公开市场操作、货币政策执行报告、新闻稿、公告、数据口径解释
- 获取方式：**爬取**（栏目列表页→详情页），部分栏目可 **RSS（若页面提供）**
- 入口类型：官网“公告/新闻/报告”栏目；建议做增量抓取（按发布时间）

2) **国家统计局（NBS）**
- 内容：CPI/PPI/PMI（部分）、工业/投资/消费、国民经济运行发布稿
- 获取方式：**爬取**（统计数据/新闻发布稿）、部分数据可从其“数据查询/表格”页面**下载/解析**
- 入口类型：数据发布日历 + 指标详情页

3) **国家外汇管理局（SAFE）**
- 内容：外汇收支、跨境资金流动、政策文件、新闻发布
- 获取方式：**爬取**（政策文件/数据与新闻栏目）
- 入口类型：政策法规/数据发布/新闻稿

4) **中国证监会（CSRC）**
- 内容：监管规则、行政处罚、审核动态、新闻发布
- 获取方式：**爬取**
- 入口类型：规则发布、行政处罚决定、新闻发布会稿件

5) **国家金融监督管理总局（NFRA，���银保监体系）**
- 内容：监管规则、处罚信息、统计与监管动态
- 获取方式：**爬取**
- 入口类型：政策文件/处罚通报栏目

6) **财政部（MOF）/国务院政策文件**
- 内容：财政政策、政府债务相关政策、重要政策文件
- 获取方式：**爬取**
- 入口类型：政策文件公开栏目

## B. 中国：市场公告与披露（交易所/信息披露）
7) **上海证券交易所（SSE）公告与规则**
- 内容：上市公司公告、监管规则、纪律处分、问询/监管动态
- 获取方式：以**爬取/解析披露页面**为主；部分栏目可能有**接口/JSON**（视页面请求而定）
- 入口类型：公告检索/规则发布/监管动态

8) **深圳证券交易所（SZSE）**
- 内容同上
- 获取方式：**爬取/接口（看页面请求）**
- 入口类型：信息披露/公告查询/监管信息公开

9) **北京证券交易所（BSE）**
- 内容：北交所公告、规则、监管动态
- 获取方式：**爬取**
- 入口类型：信息披露栏目

10) **巨潮资讯（CNINFO，A股官方披露聚合）**
- 内容：A股公告、定期报告、临时公告
- 获取方式：通常可通过其公开检索页面的**接口请求（JSON）**或**爬取**实现（注意限速与使用条款）
- 入口类型：公告检索（按证券代码/关键词/日期）

> 如果你只做一个“公司公告”源，CNINFO 往往是性价比最高的聚合入口（但务必限速+缓存）。

## C. 美国/全球：宏观、央行与官方数据（强烈推荐）
11) **FRED（Federal Reserve Economic Data）**
- 内容：海量宏观时间序列（利率、通胀、就业、金融条件等）
- 获取方式：**官方 API（强烈推荐）**
- 备注：注册免费 API key；适合做你自己的宏观数据库底座

12) **美联储（Federal Reserve / FOMC）**
- 内容：声明、会议纪要、点阵图相关材料、讲话稿
- 获取方式：**爬取**（网页结构稳定）
- 入口类型：FOMC 会议页面 + 新闻稿/讲话

13) **美国劳工统计局（BLS）**
- 内容：CPI、就业（非农相关）、工资、行业就业等
- 获取方式：**官方 API**（常用指标可直接拉）
- 入口类型：Public Data API

14) **美国经济分析局（BEA）**
- 内容：GDP、PCE、国际收支等
- 获取方式：**官方 API**
- 入口类型：BEA API

15) **美国财政部（Treasury）**
- 内容：国债收益率曲线、拍卖结果、财政数据（部分）
- 获取方式：很多数据有**公开数据接口/下载**；也可**爬取**公告新闻
- 入口类型：Daily Treasury Yield / auction data 等

16) **SEC EDGAR（美国公司披露）**
- 内容：10-K/10-Q/8-K 等
- 获取方式：**官方 API/下载（强烈推荐）**（SEC 有结构化接口与文件下载）
- 备注：注意 SEC 对 UA/限速/礼貌抓取要求非常严格（必须设置 User-Agent、限速）

17) **ECB（欧洲央行）**
- 内容：��率决议、纪要、讲话、统计数据
- 获取方式：**爬取**（新闻与文件）+ 其统计数据有**API/SDMX**可用
- 入口类型：Press releases / Statistics

18) **BoE（英央行）**
- 内容：利率决议、报告、统计
- 获取方式：**爬取** + 部分数据**下载/API**
- 入口类型：News/Statistics

19) **BIS（国际清算银行）**
- 内容：季度评述、金融稳定、跨境银行统计（部分）
- 获取方式：**下载/爬取**（报告 PDF、数据表）
- 入口类型：Publications / Statistics

20) **IMF Data / World Bank Data / OECD Data**
- 内容：跨国宏观、债务、贸易、发展指标
- 获取方式：多为**API/下载**（IMF 常见 SDMX；World Bank 有简单 REST）
- 入口类型：各自数据门户

## D. 大宗与能源（公开数据）
21) **EIA（美国能源信息署）**
- 内容：原油库存、产量、能源价格、周报/月报数据
- 获取方式：**官方 API**
- 入口类型：EIA Open Data API

22) **IEA（国际能源署）**
- 内容：部分报告与数据（不少需订阅，但也有公开内容）
- 获取方式：**爬取/下载**公开报告（注意付费墙）
- 入口类型：Reports / News
